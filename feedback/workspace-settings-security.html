Workspace policy

Private: The workspace is inaccessible to outsiders. Only workspace members can access the workspace.

Secret: The workspace is hidden from outsiders. Only workspace members know it exists.

Open: The workspace can be explored by outsiders. Outsiders can browse but not actively participate.

Self-managed: Outsiders can self-join becoming a workspace member.

Admin-managed: Only administrators can add workspace members.

Team-managed: Workspace members can add outsiders as a workspace member.

Moderate: Workspace members can do everything: read, add, publish and edit content.

Publish: Workspace members can read, add and publish content. They cannot edit other member's content.

Produce: Workspace members can read and add content. They neither publish nor edit content.

Consume: Workspace members can read content. They cannot add, publish or edit content.

The participant policy is the default permission settings for all members of this workspace. Execptions for individual people may be made on the Members tab.
